# web_timesheet_editer 要件定義

最終更新: 2026-05-01

## 1. プロダクト概要

日本のアニメーション制作向け、ブラウザで動作するタイムシートエディタ。
TDTS（東映デジタルタイムシート）/ XDTS（exchange Digital Time Sheet）両フォーマットの読み書きに対応し、Edit / Preview / Template 設定の3モードで構成される。

---

## 2. アプリケーションモード

最上段のタブで切替（独立配置）。

| モード | 役割 |
|---|---|
| **Edit** | データの入力・編集（現メイン画面） |
| **Preview** | テンプレート画像へのデータ流し込み・手書き書き込み・画像/PSD出力 |
| **Template 設定** | プレビュー用ベース画像と印字座標の管理 |

モードに応じてメニューバー一部（「挿入」など）を切替。共通項目: ファイル / 編集 / 表示 / 設定 / ヘルプ。

## 4. シート管理（VERSIONフィールド）

VERSIONフィールドをドロップダウン化し、シート切替・追加・編集を統合。

### 4.1 シート追加モーダル（東映DigitalTimeSheet 3.1.0準拠）

```
シート名:    [sheet3      ▼]  ※履歴選択可能
シート色:    [           ▼]
引継ぎシート: [sheet2      ▼]
データ:
  ⦿ すべて（タイムテーブル+補助データ全部）
  ○ シートデータ（タイムテーブルのみ）
  ○ 補助データ（メモ・注意コマ・セル置き換え・ブックデータ）
  ○ 新規（引継ぎなし）
```

### 4.2 制約
- 最大20枚
- 右クリックで「シート名変更 / シートコピー / シート削除」
- タブのドラッグ&ドロップで並び順変更（Phase H で対応）

### 4.3 メタデータのカット共有

設定 → エディタ環境設定 → 共有メタデータ：

| 項目 | デフォルト |
|---|---|
| TITLE | 共有 |
| EPISODE | 共有 |
| SCENE | 共有 |
| CUT | 共有 |
| TIME（秒・コマ） | 共有 |
| NAME | 共有 |
| DIRECTION | 個別 |
| VERSION | 常に個別（シート識別） |
| PAGE | システム自動計算 |

共有項目は変更が全シートに即反映、個別項目はそのシートのみ。

## 5. ファイルフォーマット

### 5.1 TDTS
- プレフィックス: `toeiDigitalTimeSheet Save Data\n`
- フィールド: ACTION(0) / SOUND(3) / CELL(4) / CAMERA(5)
- `fontColorId` あり（6色パレット）
- 1ファイルに複数シート格納（`timeSheets[]`）

### 5.2 XDTS
- プレフィックス: `exchangeDigitalTimeSheet Save Data\n`
- フィールド: CELL(0) / SOUND(3) / CAMERA(5) ※原画/動画区別なし
- カメラ values: `[指示, X, Y, 拡大率, 回転, 中心X, 中心Y]`（最大7要素）
- 1ファイルに複数シート格納（`timeTables[]`）

### 5.3 互換変換ルール

**XDTSインポート時**
- CELLフィールド → 取込先選択（ACTION / CELL / 両方）
- カメラは指示文字列のみ取り出し、内部Kindに推定マップ。X/Y等の数値は捨てる

**XDTSエクスポート時**
- ACTION + CELL の出力ポリシー選択（ACTIONのみ / CELLのみ / マージ）
- カメラ X/Y 等は0埋め（情報欠落の旨ダイアログ警告）

### 5.4 インポート/エクスポート選択ダイアログ

```
取込/出力する項目:
  ☑ メタ情報
  ☑ DIRECTION
  ☑ ACTION (原画)
  ☑ SOUND  (セリフ)
  ☑ CELL   (動画)
  ☑ CAMERA (撮影)
  ☑ BOOK
  ☑ ストロボ自動マージ

既存データの扱い: ⦿上書き ○マージ ○追加
```

## 6. プレビューモード

### 6.1 ツールパネル

フローティング、ドラッグ可能、シンプルSVG線画アイコン。

| パネル | 機能 |
|---|---|
| ペン | 太さ、色、Photoshop互換ショートカット |
| 消しゴム | 太さ |
| 選択 | ボックス（M）/ 投げ縄（L）切替 |
| 色 | 現在色表示、パレット・スポイト |

### 6.2 レイヤー構造（PSD出力時）

| レイヤー | 内容 |
|---|---|
| Memo（最上面） | ユーザーペン手書きストローク（透過） |
| Data | 自動生成テキスト・カメラ記号・ストロボ等（透過） |
| Template | 白紙シート画像 |
| Background | 白ベタ |

### 6.3 ページネーション

- データが行数超過 → 縦方向ページ追加（72/144コマ等）
- カラム超過 → 横方向ページ追加
- Page欄に `1/3`, `2/3` 連番自動印字
- DIRECTION・BOOK は「1ページ目のみ」印字

### 6.4 BOOK欄のACTION連動

Template設定でBOOK枠は廃止。ACTIONグリッド枠から自動位置計算。

### 6.5 Direction自動改行

Direction欄が右側のBOOKに衝突する手前で Word Wrap 自動改行。

## 7. Templateモード

### 7.1 機能
- ベース画像アップロード（JPG / PNG）
- 印字項目のON/OFFトグル
- バウンディングボックス指定（ドラッグ&リサイズ赤半透明ガイド）
  - 単一枠: Title, Episode, Scene, Cut, Time, Name, Direction, Page
  - グリッド枠: ACTION, SOUND, CELL, CAMERA
- グリッド詳細: 行数・列数で等分割
- IndexedDB 保存（画像+座標+ON/OFF）

## 10. 自動保存・未保存警告

- **デバウンス方式**：操作後3秒アイドルで localStorage に保存
- pushHistory時に `isDirty=true`、保存時に `isDirty=false`
- `beforeunload` で `isDirty===true` なら警告ダイアログ
- 復元: ブラウザ再起動時に「前回のセッションを復元しますか？」モーダル

---

## 11. 国際化

- `js/i18n.js` に辞書 `{ ja: {...}, en: {...} }` 集約
- 全UIラベルは `t('key.path')` 経由で取得
- 言語選択は localStorage 保存
