# Web Timesheet Editor (Beta)

アニメーション制作用タイムシートをブラウザ上で編集できるWebアプリケーションです。

**[デモページ](https://yoda-y.github.io/web-timesheet-editor/)**

## 対応ファイル形式

| 形式 | 読み込み | 保存 |
|------|----------|------|
| TDTS (toeiDigitalTimeSheet) | ○ | ○ |
| XDTS (exchangeDigitalTimeSheet) | ○ | ○ |

## 主な機能

### エディタモード
- セル編集（動画番号、記号 ●○× など）
- 複数セル選択・ドラッグ移動・コピー
- セリフブロック・カメラワークブロック編集
- コマ挿入・削除（選択列/全レイヤー）
- レイヤー（列）追加・削除
- リピート・振り返し・ランダム振り返し
- Undo/Redo
- 複数シート対応
- 兼用カット対応（複数カットを1ファイルで管理）
- ドキュメントタブ（複数ファイル同時編集）

### プレビューモード
- A3縦テンプレートによるタイムシート表示
- 手書きレイヤー
  - ペンツール（3サイズ）
  - 消しゴムツール（3サイズ）
  - 矩形選択・投げ縄選択
  - 変形（移動・拡縮）
  - ハンドツール（パン）
  - Spaceキー押下中の一時ハンド
- ズーム・パン操作（マウスホイール対応）
- PNG/JPG/PSD書き出し（150/300dpi）
- 手書きPNG読み込み・書き出し

### その他
- 自動保存（ブラウザストレージ）
- ダーク/ライトテーマ切替
- 日本語/英語切替
- キーボードショートカット（カスタマイズ可）
- 入力色パレット（6色）
- 設定のインポート/エクスポート

## 使い方

1. [デモページ](https://yoda-y.github.io/web-timesheet-editor/)を開く
2. 「ファイル」→「開く」または「フォルダから開く」でファイルを読み込み
3. セルをクリックして編集、プレビューモードで確認
4. 「ファイル」→「保存」または「名前を付けて保存」で出力

### フォルダから開く（推奨）
「ファイル」→「フォルダから開く」を使用すると：
- 手書きデータ（PNG+INI）の自動読み込み・保存
- 同一フォルダへの上書き保存が可能

## 動作環境

- **推奨**: Chrome / Edge（File System Access API対応）
- Firefox / Safari（一部機能制限あり）

## ローカルで実行

```bash
git clone https://github.com/yoda-y/web-timesheet-editor.git
cd web-timesheet-editor
# 任意のローカルサーバーで起動
npx serve .
```
