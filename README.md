# Web Timesheet Editor (Beta)

アニメーション制作用タイムシートをブラウザ上で編集できるWebアプリケーションです。

**[デモページ](https://yoda-y.github.io/web-timesheet-editor/)**

## 対応ファイル形式

| 形式 | 読み込み | 保存 |
|------|----------|------|
| TDTS (toeiDigitalTimeSheet) | ○ | ○ |
| XDTS (exchangeDigitalTimeSheet) | ○ | ○ |
| Web Timesheet Project JSON (`.wtproj.json`) | ○ | ○ |
| Web Timesheet Project HTML (`.wtproj.html`) | ○ | ○ |

## 主な機能

### エディタモード
- セル編集（動画番号、記号 ●○× など）
- 複数セル選択・ドラッグ移動・コピー
- セリフブロック・カメラワークブロック編集
- コマ挿入・削除（選択列/全レイヤー）
- レイヤー（列）追加・削除
- リピート・振り返し・ランダム振り返し
- Undo/Redo
- 複数シート対応
- 兼用カット対応（複数カットを1ファイルで管理）
- ドキュメントタブ（複数ファイル同時編集）

### プレビューモード
- A3縦テンプレートによるタイムシート表示
- **外部テンプレート (v0.8.0〜)**
  - 任意の用紙画像にBBoxで領域を定義してデータを流し込み
  - BBox編集モーダルで視覚的に配置
  - JSONエクスポート/インポートで共有可能
  - カスタム項目 (custom1-4) で独自フィールドを追加
- 手書きレイヤー
  - ペンツール（3サイズ）
  - 消しゴムツール（3サイズ）
  - 矩形選択・投げ縄選択
  - 変形（移動・拡縮）
  - ハンドツール（パン）
  - Spaceキー押下中の一時ハンド
- ズーム・パン操作（マウスホイール対応）
- PNG/JPG/PSD書き出し（150/300dpi）
  - PSDは background / template / data / memo の4レイヤーに分離
  - ResolutionInfo (DPI) 埋め込み済みでクリスタ等でも実寸表示
- 手書きPNG読み込み・書き出し

### その他
- 自動保存（ブラウザストレージ）
- ダーク/ライトテーマ切替
- 日本語/英語切替
- キーボードショートカット（カスタマイズ可）
- 入力色パレット（6色）
- セリフタイプ (normal/off/mono/背) 対応
- 設定のインポート/エクスポート

## プロジェクトファイル (v0.9.0〜)

Web Timesheet Editor 専用形式として `.wtproj.json` / `.wtproj.html` をサポート。
全エディタ状態（メタ・セル・ブロック・customFields・dialogueType・外部テンプレ・手書き画像/ストローク）を1ファイルに保存できます。

- **`.wtproj.json`**: テキスト形式。差分管理や軽量配布向け
- **`.wtproj.html`**: 自己完結型HTML。ブラウザで直接開けるランチャー付き
  - 現状はランチャーから「JSONをダウンロード」または「アプリを開く」操作で読込（HTMLからアプリへの自動転送は今後対応予定）

## 既知の制限

- `.wtproj.html` を直接開いた場合、ランチャー経由でJSONダウンロードまたはアプリを開く操作が必要
- `Ctrl+S` のプロジェクト形式保存への切替は未実装
- XDTSではセリフタイプ・外部テンプレ紐付け・カスタム項目は保持されません（書出時に警告）
- 外部テンプレートライブラリはブラウザ内 (IndexedDB) 保存。共有はJSONエクスポートで対応
- Firefox / iPad Safari はFile System Access API非対応のため、上書き保存や手書きPNG/INI自動保存はダウンロード保存になります

## 使い方

1. [デモページ](https://yoda-y.github.io/web-timesheet-editor/)を開く
2. 「ファイル」→「開く」または「フォルダから開く」でファイルを読み込み
3. セルをクリックして編集、プレビューモードで確認
4. 「ファイル」→「保存」または「名前を付けて保存」で出力

### フォルダから開く（推奨）
「ファイル」→「フォルダから開く」を使用すると：
- 手書きデータ（PNG+INI）の自動読み込み・保存
- 同一フォルダへの上書き保存が可能

## 動作環境

- **推奨**: Chrome / Edge（File System Access API対応）
- Firefox / Safari（一部機能制限あり）

## ローカルで実行

```bash
git clone https://github.com/yoda-y/web-timesheet-editor.git
cd web-timesheet-editor
# 任意のローカルサーバーで起動
npx serve .
```
